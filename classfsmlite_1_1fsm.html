template<class Event >
state_type fsmlite::fsm< Derived, State >::no_transition ( const Event &  event)
inlineprotected

Called when no transition can be found for the given event in the current state. Derived state machines may override this to throw an exception, or change to some other (error) state. The default is to return the current state, so no state change occurs.
